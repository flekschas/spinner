# Spinner
~~CSS3~~ SMIL-animated spinner using SVG. Originally I animated SVG exclusively with CSS3 but the browser support is worse compared to SMIL.

## Example
See the spinner spinning: https://flekschas.github.io/spinner/

## Browser Support

### SMIL

#### Theoretical

* SVG: http://caniuse.com/#feat=svg-smil

**Desktop** 
* Chrome >= 5
* Safari >= 6
* Firefox >= 4
* Opera >= 10.1

**Mobile**
* Mobile Safari >= 6.1
* Android Browser >= 3

#### Tested

**Desktop**
* Chrome 19 [1], 44 [1]
* Safari 6.2.7 [2]
* Firefox 5 [2], 36 [1]
* Opera 12.15 [2], 28 [1]

**Mobile**
* Mobile Safari 8 [1]
* iOS Chrome 42 [1]
